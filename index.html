Maps are also now searchable. Select the magnifying glass icon in the top left corner of the map and enter a location of interest.

A New Chapter in Story Points.

While not one of the marque features of the masses, Story Points in Tableau 9.0 has a few updates worth mentioning.

No longer are developers constrained to a one size fits all navigation block. Developers now have the ability to customize the size of navigation blocks. While it may not be recommended, developers can make navigation blocks the size of a laptop screen! Additionally, developers have the freedom to control text size, fonts, and colors within navigation blocks.

Go forth and tell your story.

Drag and Drop Analytics

Tableau 9.0 has added an entirely new analytics pane. The new analytics pane allows developers a set of features to be dragged and dropped, just as with any other pill.

Just think, developers no longer have to navigate dialog boxes to add reference lines. Better yet, these reference and trend lines are dynamic, meaning references update based on user selections. We can't wait to see how you put them to use!
